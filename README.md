## BoltMQ是什么
* 是一个分布式发布订阅的消息系统 
## BoltMQ现在有哪些特性
* 负载发送消息,除顺序消息之外所有的消息轮循发送到所有消息处理机器  
 
* 顺序写,适应磁盘特性  
 
* 使用内存映射
 
* 集群消费

* 保证消息至少投递一次
 
* 支持同步消息,单向消息,异步消息,顺序消息的发送方式
